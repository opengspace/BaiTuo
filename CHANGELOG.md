# CHANGELOG

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- **小地图功能** - 拜托小镇右上角显示小地图，快速查看小人位置，点击快速移动视角
  - 显示整个小镇缩略图（街道网格、草地）
  - 角色位置用彩色圆点标识（根据情绪：快乐=金色、正常=绿色、焦虑=橙色、愤怒=红色）
  - 白色矩形框显示当前视口位置
  - 点击小地图任意位置平滑滚动视角到该位置（500ms动画）
  - 点击角色圆点快速定位到该角色
- **AI 配置功能** - 设置页添加 AI 配置入口，支持多厂商预设
  - 支持 OpenAI、Anthropic 等主流厂商
  - 可配置 API Key 和自定义端点
- **角色性格系统** - 拜托人角色自动初始化个性化性格，影响抱怨内容风格

### Changed
- **小镇角色显示** - 所有拜托人都显示在小镇，按完成/取消/未完成情况计算心情
  - 显示所有拜托过需求的人（包括历史已完成）
  - 无拜托人名称的统一合并显示为"佚名"
  - 情绪计算基于完成率、逾期情况、取消情况
  - 点击角色显示所有历史待办（未完成 + 已完成 + 已取消）
- **四象限矩阵视图像素化** - 与拜托小镇视觉风格统一，告别千篇一律的待办应用 UI
  - 象限头部改为 RPG 招牌样式（像素边框 + ZCOOL KuaiLe 像素字体 + 黑底白字计数方块）
  - 待办卡片改为像素告示牌风格：阶梯 box-shadow 像素边框、左侧象限色条、像素金币信誉值
  - 新增 6 个 8×8 像素 SVG 图标（PixelClock/User/Check/More/Plus/Coin）替换 lucide 线性图标
  - 难度标签改为纯色像素方块，完成按钮为绿色像素按钮
  - 拖拽 DragOverlay 带像素残影（scale + 微旋转）
  - 象限空状态用像素虚线边框，拖拽悬停时像素闪烁高亮
  - 保留 dnd-kit 完整拖拽与排序能力，无功能回退
- **全应用像素化扩展** - 侧边导航、顶部 Header、设置弹窗统一像素风，通用组件升级带动全应用风格统一
  - 通用组件 Button / Switch / Modal 像素化：方形边框、`font-pixel` 字体、按压反馈、方形 Switch thumb
  - 侧边栏：导航项激活态黑底白字像素方块、象限色点改方块、统计块像素内边框、成就按钮 RPG 风格
  - 顶部 Header：像素边框分隔、信誉值徽章变 primary 实色像素方块、操作按钮全换像素图标
  - 设置弹窗：4 个分区卡片用 `pixel-border-inset`，range slider / input / select 均用像素样式（含聚焦双层高亮、像素三角下拉箭头、WebKit + Firefox 滑块样式）
  - 新增 15 个 8×8 像素 SVG 图标（PixelHome/Grid/List/Cal/Check2/X/Trophy/Download/Upload/Gear/Warn/Spark/Eye/EyeOff/XClose）
  - 新增 `.pixel-input` / `.pixel-select` / `.pixel-range` CSS 工具类
- **列表视图与剩余页面像素化** - 补齐 list/today/completed/cancelled 四视图、待办表单、信誉卡、成就明细页
  - HomePage 内联列表：卡片 `pixel-border`、列表项 `pixel-border-inset`、空状态 `pixel-dashed`、标题与数值 `font-pixel`
  - ReputationCard + AchievementList：像素边框、`pixel-border-inset` 进度条与统计块、成就项像素方块、lucide 图标全换 Pixel
  - TodoForm：input/textarea 用 `pixel-input`、象限/难度选择按钮 `pixel-border`、信誉预览 `pixel-border-inset`
  - ReputationDetailPage：header 像素边框、汇总卡 `pixel-border`、记录项 `pixel-border-inset`、筛选 `pixel-select`、分页按钮 `pixel-border`
  - 新增 7 个像素图标（PixelFlame/Target/Trending/Chevron/ChevronLeft/ArrowLeft/Star）

## [0.3.1] - 2025-05-26

### Fixed
- 拜托小镇画布尺寸过小，大屏显示不全，扩展为视窗 2 倍并默认居中展示
- 增加街道密度保持草地区域大小不变，避免一眼看过去只有一个大区域

## [0.3.0] - 2025-05-25

### Added
- **拜托小镇视图** - 像素风格小镇地图，将待办可视化为像素小人在地图中生活
- **角色情绪系统** - 根据逾期情况自动计算情绪（开心/平静/难过/气愤），不同情绪有不同动画
- **抱怨系统** - 逾期任务的拜托人会生成抱怨，点击气泡可查看内容
- **像素字体** - 引入 ZCOOL KuaiLe 像素风格字体
- **小镇地图** - 街道网格将小镇分为 9 个草地区域，每区域 6 栋房子，沿边缘种树
- **左上角介绍** - 黑色半透明覆盖层，随机展示拜托哲学短句
- **拖拽地图** - 支持拖动探索小镇，角色跟随地图移动

### Changed
- **请求人 → 拜托人** - 全局文案从"请求人"改为"拜托人"
- **侧边栏排序** - 拜托小镇移至导航栏第一位
- **全屏展示** - 拜托小镇视图去除 padding，内容占满整个区域

### Fixed
- 修复屋顶绘制为倒三角的问题
- 修复角色坐标未跟随地图偏移的问题
- 修复名字标签视觉偏移，精确居中

## [0.2.1] - 2025-05-20

### Changed
- **逻辑删除** - 删除待办改为设置 cancelled 状态，而非真实删除数据
- **取消任务惩罚** - 取消任务时触发信誉扣分，并显示扣分动画

### Added
- **已取消视图** - Sidebar 新增已取消任务视图入口
- **恢复功能** - 已取消任务可点击恢复，重新变为待处理状态

### Fixed
- 左侧导航栏高度约束，成就入口无需滚动即可看到
- 四象限视图整体布局高度约束，每个象限独立滚动
- 成就明细页面布局高度约束

## [0.2.0] - 2025-05-19

### Added
- **成就明细页面** - 新增独立页面查看成就增加减少明细，支持类型筛选和分页
- **每日检查机制** - 自动检查逾期任务并扣除成就点数
- **逾期扣分规则配置** - 支持配置每小时扣分、每日上限、逾期天数阈值、象限系数
- **取消任务惩罚配置** - 设置弹窗支持配置取消任务的扣分规则
- **设置弹窗** - Header 添加设置按钮，可配置每日检查规则
- **四象限创建时间显示** - 任务卡片显示创建时间（相对时间格式）
- **Sidebar 成就入口** - 左侧导航栏底部添加成就明细入口按钮

### Fixed
- 设置弹窗添加高度约束，防止内容溢出
- 左侧导航栏固定不随右侧内容滚动
- 四象限视图每个象限独立垂直滚动

## [0.1.0] - 2025-05-08

### Added
- **四象限视图** - 艾森豪威尔矩阵布局，重要紧急四象限分类
- **待办管理** - 创建、编辑、删除、完成待办
- **拖拽排序** - 象限内任务拖拽排序，跨象限拖拽移动
- **信誉系统** - 完成任务获得信誉值，根据象限和难度计算奖励
- **等级系统** - 18个等级从"初来乍到"到"传说"，信誉值累积升级
- **成就系统** - 10种成就，完成数量、连续天数、时间管理等解锁条件
- **数据存储** - IndexedDB 本地存储，支持导入导出
- **多视图切换** - 四象限、全部待办、今日待办、已完成视图
- **象限筛选** - 按象限筛选待办列表
